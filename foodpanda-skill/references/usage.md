# Foodpanda AI 外送點餐助理 Skill

一個用於教學與實際應用的 **AI 半自動點餐助理**，幫使用者在叫外送前做好決策。

---

## 功能特色

**可以做的事：**
- 根據人數、預算、地區、偏好智慧推薦餐點
- 自動分配預算（主餐 + 配菜 + 飲料）
- 多人團訂規劃與份量估算
- 產出結構化的建議點餐清單
- 支援互動式問答模式
- 支援自然語言輸入（如：「2個人300元板橋不要辣」）
- 輸出 JSON 格式，方便串接其他系統

**不做的事：**
- 不自動付款或下單
- 不保存帳號密碼
- 不繞過登入驗證
- 不保證平台資料即時正確

## 快速開始

### 環境需求
- Python 3.8+（無需安裝額外套件）

### 執行方式

**1. 示範模式（一次跑完所有測試案例）**
```bash
python foodpanda_helper.py --demo
```

**2. JSON 輸入模式**
```bash
python foodpanda_helper.py '{"location":"新北市板橋區","budget":300,"meal_time":"dinner","category":["便當"],"avoid":["炸物"],"people":2,"drink":true,"fast_delivery":true,"chain_ok":true}'
```

**3. 互動式問答**
```bash
python foodpanda_helper.py --interactive
```
然後輸入自然語言：
```
> 2個人300元板橋晚餐不要辣
```

**4. JSON 輸出（串接用）**
```bash
python foodpanda_helper.py --json '{"location":"新北市板橋區","budget":200,"people":1}'
```

## 輸入參數說明

| 參數 | 類型 | 預設值 | 說明 |
|------|------|--------|------|
| `location` | string | `""` | 地區（如「新北市板橋區」） |
| `budget` | int | `200` | 總預算（元） |
| `meal_time` | string | `""` | 時段：breakfast/lunch/dinner/late_night/afternoon_tea |
| `category` | list | `[]` | 偏好類型（如 `["便當","湯品"]`） |
| `avoid` | list | `[]` | 禁忌（如 `["辣","炸物"]`） |
| `people` | int | `1` | 用餐人數 |
| `drink` | bool | `false` | 是否加點飲料 |
| `fast_delivery` | bool | `false` | 是否限快速送達 |
| `chain_ok` | bool | `true` | 是否接受連鎖店 |

## 教學應用建議

此 Skill 適合用於以下教學場景：

### 課堂流程（建議 4 段式）

1. **問題定義**：AI 能不能直接幫你點外送？哪些步驟適合 AI 先做？
2. **輸入設計**：如何把自然語言轉成結構化 JSON 條件
3. **規則引擎**：預算分配、偏好篩選、禁忌排除的程式邏輯
4. **輸出應用**：產出可執行的點餐清單，而非空泛建議

### 延伸方向

| 方向 | 技術 | 難度 |
|------|------|------|
| 串接 LINE Bot | LINE Messaging API | 中 |
| 串接 Google Sheet 菜單 | Google Sheets API | 低 |
| Web UI 介面 | HTML + JS | 低 |
| 多代理模式 | Claude / OpenAI Agent | 高 |
| 接真實平台資料 | Playwright 爬蟲 | 高 |
