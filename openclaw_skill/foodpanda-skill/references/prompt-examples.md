# Foodpanda Assistant - 提示詞範例集

> 以下範例可直接用於教學展示，讓學員體驗 AI 外送助理的互動流程。

---

## 基本用法

### 簡單推薦
```
今天午餐吃什麼？
```

### 指定預算
```
幫我推薦 200 元內的晚餐
```

### 指定人數與預算
```
幫我推薦 2 個人、300 元內的午餐
```

---

## 進階條件

### 指定地區 + 偏好
```
我在板橋，想吃有湯的晚餐，預算 200 元
```

### 排除特定食物
```
幫我找不辣、不炸的便當，一個人 150 元內
```

### 健康需求
```
我想吃健康一點的午餐，低油不炸，預算 200 元
```

### 素食者
```
我吃素，幫我推薦板橋附近 150 元內的午餐
```

---

## 多人/團訂情境

### 小組訂餐
```
我們 4 個人想訂午餐，總預算 600 元，在板橋，有人不吃辣
```

### 辦公室下午茶
```
幫我規劃 5 人份的飲料訂單，預算 400 元，要快速送達
```

### 會議便當
```
幫我規劃 8 人份的會議便當，每人預算 120 元，要有飯有菜
```

---

## 特殊時段

### 早餐
```
推薦板橋附近的早餐，兩個人 150 元內
```

### 宵夜
```
深夜想吃東西，一個人 200 元內，想吃麵
```

### 下午茶
```
下午想喝手搖飲，3 個人 200 元內
```

---

## 進階互動

### 比較模式
```
幫我比較便當和麵類哪個比較划算，2 個人 300 元
```

### 調整建議
```
剛才推薦的太貴了，幫我換便宜一點的
```

### 加點飲料
```
剛才的午餐建議不錯，幫我每人再加一杯飲料，預算多 100 元
```

---

## 教學展示用：完整指令

### 結構化指令範例
```
請幫我規劃點餐：
- 人數：3 人
- 預算：450 元
- 地區：板橋
- 偏好：台式便當
- 禁忌：不吃辣、不要炸物
- 需要飲料：是
- 快速送達：否
請輸出完整的建議點餐清單。
```

### JSON 格式直接呼叫
```json
{
  "location": "新北市板橋區",
  "budget": 450,
  "meal_time": "lunch",
  "category": ["便當", "台式"],
  "avoid": ["辣", "炸物"],
  "people": 3,
  "drink": true,
  "fast_delivery": false,
  "chain_ok": true
}
```
